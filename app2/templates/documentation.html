Documentation

Node Dashboard

  1. Node ID - A binary id number associated with a single node.
  2. Location - The approximate location of each node.
  3. Status - The current operational status of each node.
  4. Last Update - A timestamp of the last received update from each node.

Server Dashboard

  1. Normalized Node Uptime Area Chart - A stacked area chart depicting the percentage of nodes on the beehive server currently in each uptime category at a given time. This contains seven categories: Greater than One Day, Less than One Day, Less than Six Hours, Less than One Hour, Less than Thirty Minutes, Less than Five Minutes, Less than One Minute.
  2. Server Log - This is a log of each interaction between the website and the Beehive Cassandra API.
